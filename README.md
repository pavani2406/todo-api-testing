# todo-api-testing
API Testing project using Postman on To-Do AP(Beginner-friendly way to understand how HTTP requests work in practice)

#Aim
This project demonstrates API testing using Postman on the JSONPlaceholder To-Do API. 
It helps beginners understand how HTTP requests work and how to validate API responses.

## Objective
- Learn how to perform GET, POST, PUT, DELETE operations on an API
- Validate HTTP status codes and response data
- Understand Postman concepts: Collections, Workspaces, Environments
- Build a beginner-friendly project for GitHub portfolio

## Tools & Technologies
- Postman (Web/Desktop)
- JSONPlaceholder API (Free mock API)
- Git & GitHub
